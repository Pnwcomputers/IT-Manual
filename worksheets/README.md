## üìñ Overview
This directory contains the physical (or fillable PDF) forms used during service. Unlike the digital "Procedures" in other folders, these files are designed to be printed out, attached to a clipboard, and traveled with the device as it moves through the shop. They ensure that every technician performs the same 15-point inspection, regardless of who is working that day.

## üñäÔ∏è Usage: The "Traveler" System
*How to use these worksheets effectively in the shop.*

1.  **Intake:** When a machine is checked in, the front desk prints the relevant **Work Order PDF**.
2.  **Attachment:** The sheet is clipped to the device or taped to the case.
3.  **Execution:** As the technician completes steps (e.g., "Ran Malwarebytes"), they **initial the box** on the paper.
4.  **QC (Quality Control):** Before the device leaves, a second set of eyes reviews the sheet to ensure no boxes were skipped.
5.  **Digitization:** The completed sheet is scanned or photographed and uploaded to the ticket history for liability protection.
