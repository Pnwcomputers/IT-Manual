## üì∏ Contribution Guidelines
*To keep this directory clean, please follow these rules when uploading new images:*

### 1. Naming Conventions
**Do NOT** use generic names like `image1.png` or `screenshot.jpg`.
* **Bad:** `img_5502.jpg`
* **Good:** `dell-bios-boot-menu.jpg`
* **Good:** `rj45-t568b-pinout.png`

### 2. File Formats
* **Diagrams/Text:** Use **.PNG** (Better clarity for lines and text).
* **Photos:** Use **.JPG** (Smaller file size for photographs).

### 3. Compression
Please run large images through a compressor (like TinyPNG) before uploading to keep the repository size manageable.
