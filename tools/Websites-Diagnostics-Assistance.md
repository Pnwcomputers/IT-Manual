## üì° Networks, Email, Domains, DNS, and IP Lookup

These tools are essential for diagnosing connectivity, domain resolution, and email deliverability issues.

### Network and Reliability Testing
* **Looping Ping Test:** `www.ping.canbeuseful.com` (Continuous ping testing)
* **Internet Loop Test:** `trt-netztest.at` (Network quality and loop testing)
* **Internet Reliability Testing:** `packetstats.com`

### IP and Domain Lookup
* **Public IP Address Lookup:** `whatismyipaddress.com` (Finds your external IP address)
* **Whois & Domain Reverse Engineering:** `whois.domaintools.com` (Details on domain registration)
* **Reverse IP Lookup from Domains:** `dnschecker.org/domain-ip-lookup` (Finds the IP address associated with a domain)
* **DNS Lookup/Traceroute:** `https://mxtoolbox.com/DnsLookup.aspx` (Checks DNS records and traces network paths)

### Blacklist Email Checking
These tools check if an IP address or domain is listed in common email blacklists, which can affect email delivery.
* **MXToolbox Blacklist Check:** `https://mxtoolbox.com/blacklists.aspx`
* **WhatIsMyIPAddress Blacklist Check:** `https://whatismyipaddress.com/blacklist-check`
